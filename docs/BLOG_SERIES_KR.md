# 시나브로 블로그 시리즈 (추천 4편)

## 1편. 문제 정의와 제품 설계

- 대상 사용자(탈북민)의 언어 적응 문제
- 왜 텍스트 학습만으로는 부족한지
- 학습-평가-복습 루프 설계 의도

## 2편. Flutter에서 직접 말하기 구현

- 웹/앱 마이크 권한 처리 전략
- 실시간 레벨/피치 시각화 방식
- 자동 종료 + 수동 중단/취소 UX 설계

## 3편. Django 발음 평가 API와 안정화

- 업로드 포맷 처리(m4a/webm/wav/ogg)
- 디코딩 실패 케이스와 fallback
- 점수/피드백 반환 스키마 설계

## 4편. 실서비스 배포 자동화

- Docker Compose + Nginx + HTTPS 구성
- GitHub Actions Self-hosted Runner 배포
- 운영 중 이슈 대응 루틴과 smoke check

## 각 글 공통 템플릿

1. 문제
2. 원인 분석
3. 해결 방법
4. 코드/설정 핵심
5. 검증 결과
6. 다음 개선
