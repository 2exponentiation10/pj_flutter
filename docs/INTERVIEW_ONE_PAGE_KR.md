# 시나브로 면접 요약 1페이지

## 프로젝트

- 이름: 시나브로(Onsaemiro)
- 목적: 탈북민의 남한 표준어 발화 교정
- 역할: 풀스택(Flutter + Django + 배포 자동화)

## 핵심 기여

- 직접 말하기 실시간 피드백 UX 구현(레벨/피치/STT)
- 음성 평가 파이프라인 안정화(포맷 fallback, stop/read race 완화)
- 홈/진행도/복습 큐를 실데이터 기반 대시보드로 개선
- GitHub Actions + Self-hosted Runner 배포 자동화 구축

## 기술 난제

- iOS Safari/웹 브라우저 마이크 정책 차이
- 녹음 포맷별 디코딩 실패 및 간헐적 빈 오디오
- 평가 요청 실패 시 사용자 이탈을 줄이는 오류 복구 UX

## 해결 방식

- 플랫폼별 입력 경로 분리 + 권한/진단 상태 노출
- 오디오 stop 후 읽기 재시도와 업로드 content-type 보정
- 모달 중심에서 snackbar/상세시트 중심 UX로 전환

## 결과

- 실제 도메인 운영 및 HTTPS 서비스
- 자동 배포 파이프라인으로 반복 배포 가능
- 학습/평가/복습/관리 콘솔까지 일관된 사용자 플로우 완성

## 배운 점

- 음성 기능은 모델 품질 이전에 입력 파이프라인 신뢰성이 우선
- 모바일 UX는 기능 추가보다 실패 복구 경험이 중요
- 작은 디버깅 로그/진단 패널이 운영 효율을 크게 개선
