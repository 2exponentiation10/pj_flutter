## 2) 디자인 품질(수동 점검)

### 공통
- [ ] 홈/학습/평가/복습 화면 카드 라운드/간격/텍스트 크기가 일관적이다.
- [ ] 버튼 터치 영역이 충분하고(모바일), 눌렀을 때 피드백이 자연스럽다.
- [ ] 다크 모드에서 텍스트 대비가 충분하다(제목/본문/보조문구 모두).
- [ ] 작은 화면(아이폰 미니급)에서 좌우 잘림/겹침이 없다.

### 홈
- [ ] 히어로 카드(다음 학습 챕터)가 한 화면에서 핵심 정보+CTA를 명확히 보여준다.
- [ ] 빠른 실행 타일 4개가 스크롤 중에도 식별이 쉽다.
- [ ] 통계 카드 2개가 숫자/라벨이 깨지지 않고 정렬된다.

### 학습/평가
- [ ] 상단 히어로 + 난이도 필터 + 챕터 카드의 정보 계층이 명확하다.
- [ ] 필터(전체/초급/중급/고급) 전환 시 리스트 반영이 즉시 된다.
- [ ] 챕터별 유닛 이동 버튼이 오동작 없이 각 상세 화면으로 이동한다.

### 복습 큐
- [ ] 복습 이유, 우선순위, 난이도/상황, 점수 정보가 한 카드에 읽기 좋게 배치된다.
- [ ] "이 문맥 복습하기" 클릭 시 해당 억양 학습 화면으로 정확히 이동한다.
